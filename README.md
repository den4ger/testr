# Прогнозирование общего объема рынка по регионам в денежном выражении

Веб-приложение на базе фреймворка **Streamlit** для интерактивного анализа, рекурсивного прогнозирования и оценки неопределенности общего объема рынка жилой недвижимости по регионам РФ в денежном выражении. 

Целевая переменная рассчитывается на основе реализованного спроса: `Объем рынка (млн руб.) = Цена за кв. м × Проданная площадь (кв. м)`.

## Основной функционал:

1. **Прогноз общего объема рынка по регионам:**
   * Помесячный прогноз на краткосрочный (3 месяца) и среднесрочный (6 месяцев) горизонты планирования 2025 года.
   * Использование алгоритма ** рекурсивной регрессии** (динамический пересчет лагов проданных площадей на основе предыдущих предсказаний модели).
   * Визуализация исторических данных за 2024 год, фактических значений и прогнозного тренда 2025 года.
   * **Оценка неопределенности (Prediction Intervals):** построение 95%-го доверительного интервала методом **конформных прогнозов ** на основе эмпирического распределения абсолютных ошибок рекурсии.

2. **Разведочный анализ данных (EDA):**
   * Статистический обзор объемов рынка в разрезе субъектов РФ.
   * Анализ асимметрии распределения целевой переменной и обоснование применения логарифмического преобразования (`log1p`) для стабилизации дисперсии перед обучением.
   * Интерактивная карта корреляций переменных

3. **Верификация и метрики качества:**
   * Сравнение результатов 5 моделей: `Linear Regression`, `Ridge`, `Median Baseline`, `CatBoost` и `CatBoost_log` (лучшая конфигурация).
   * Автоматическая интерактивная подсветка лучших метрик ($R^2$, MSE, MAE, RMSE, MAPE).
   * Интерактивный график важности признаков для интерпретации ключевых макроэкономических и локальных факторов, влияющих на рынок.
